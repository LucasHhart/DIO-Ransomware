# Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio prático da plataforma DIO, com o objetivo de aplicar conceitos de Segurança da Informação, Criptografia e Programação em Python através da implementação de um ransomware educacional em ambiente controlado.

A proposta do desafio consiste em criar dois scripts principais:

encrypter.py → responsável por criptografar arquivos;
decrypter.py → responsável por descriptografar os arquivos previamente criptografados.

O projeto foi desenvolvido exclusivamente para fins de aprendizado, permitindo compreender na prática como funciona o processo de criptografia utilizado por ransomwares reais.

# Objetivos do Projeto
- Aplicar conceitos de criptografia utilizando Python;
- Compreender o funcionamento básico de um ransomware;
- Manipular arquivos utilizando a linguagem Python;
- Trabalhar com criptografia simétrica AES;
- Desenvolver habilidades em Segurança Ofensiva e análise de ameaças;
- Documentar tecnicamente o projeto utilizando GitHub.

# Tecnologias Utilizadas
Python 3
Biblioteca pyaes
Kali Linux
